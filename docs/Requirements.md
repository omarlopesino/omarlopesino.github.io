This document aims to write all we need in our site. It will work as an initial list of requirements that probably can be evolved.
## Posts

The main idea of this page is to share my knowledge. Not only that, but use it as a way to encourage me to do small side projects that could help others.

## About me

I want to show a professional profile about me: what I do, in what projects I collaborate, and my knowledge.
## Components

The page needs components to be able to construct:
- A blog posts list
- An about me
- A post detail
## Multilingual

I want the blog servers to any person in the world so that is why the main language is english. Of course, as a spanish native speaker, I want also dedicate some time to make it easier to read for those who speak spanish around the world.

So, two languages: spanish and english.
### Pages

We need to show through different pages:
- A small presentation about me.
* Blog

It is pending to decide to show first the about me or the blog entries. A small presentation sounds cool. 
## Search

### External

Since the first moment the site must be ready to be found externally. The idea is that people should be able to find the page by google / LLM searchs .

Basically, a SEO strategy. URL structure, content,  metatags sed, sitemap, robots and llms txt must be creating according to this objective.
### Internal

When the number of posts evolve ina  blog we need some tools to let users find specific content, so the site should allow minimal search:
- By body content.
- By tags

It does not need to be complex for an MVP.

## Feed

For interested users, the site will provide a way to subscribe to content. With subscription we mean that user must have a way to get notified to get new content. It must not imply  saving its data.

## Data

No data from users will be saved. No contact forms, no analytics, no cookies.  The action needed on the project is to never solve anything by saving users data.

## Engage

Minimal recognition about the autor (myself) would be good. So, there must be some way to self-promote in the posts.

Ideas: a hero in middle of content? A hero at the end of the content?
